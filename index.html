How the system works

The Jetson Nano reads soil moisture, moves a 3D-printer-style carriage along a rail, and activates a small pump to water only the spots that need it.

Jetson Nano control Single-axis gantry Moisture-based rules Expandable zones
  • Sample each moisture sensor on a schedule.
  • Calculate which plant positions are under target moisture.
  • Move the rail carriage to that position.
  • Enable the pump for a short, metered watering pulse.
  • Repeat until all dry spots are topped up.

Roadmap & future plans

Start as a simple prototype, then turn it into a polished, easy-to-install system others could buy and drop into their grow space.

  • Phase 1 — Basic prototype on a single rail.
  • Phase 2 — Web dashboard for manual / auto control.
  • Phase 3 — Tidy wiring harness & printable mounts.
  • Phase 4 — “Kit” version and simple install guide.
